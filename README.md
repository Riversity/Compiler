# Mx* Compiler

This project implements a compiler to RISC-V/32 from a custom, Java-like language.

`src/parser` is the parser file generated by ANTLR from `Mx.g4`.

`src/sema`builds the AST and analyses the semantics.

`src/ir` compiles the tree structure into LLVM.

`src/codegen` compiles LLVM IR into assembly.
